Detailed Description

Base Types - designed to be directly compatible with OpenGL.

Base node type structure that is used for our Scene Graph.
Global Tree Graph used to store all of the app data structers. Multiple instances supports independent user environments and datasets. A caveat of this approach is that the app can draw its own structure.
